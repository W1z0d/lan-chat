# lan-chat
!!VIBECODED!!
возможен вход как с браузера, так и с терминала (через client.py)

у клиента есть команды:
all - показать все сообщения, которые были отправлены в текущий сеанс
new - показатаь все сообщения, которые еще не были прочитаны на этом клиенте
send - отправить сообщение
auto - включить/выключить автоматическое обновление сообщений
exit (ctrl+c) - выход
!!ВАЖНО!! в одной строке с командой не должно ничего быть - то есть "send 123" - некорректно, правильно будет send (новая строка) 123
